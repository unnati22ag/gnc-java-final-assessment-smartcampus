# gnc-java-final-assessment-smartcampus
This repo is for completion of  final assignment -smartcampus


## 📝 Problem Statement

Managing students, courses, and enrollments in a campus can be complex.
This project is a Java console application that simulates a Smart Campus Management System.
It demonstrates Object-Oriented Programming (OOP), collections, exception handling, and multithreading in a simple, easy‑to‑understand way.

## ✨ Features
### Student Management

   Add new students with ID, name, and email

   View all registered students

### Course Management

Add new courses with ID, name, and fee

Validate course fee using custom exceptions

### Enrollment System

 Enroll students in multiple courses

Store enrollments using HashMap

View all enrollments

### Menu Driven Program

Interactive console menu for all operations

### Exception Handling

Handle invalid inputs gracefully

Custom exception for negative or invalid course fees

 ###  Multithreading

Simulate enrollment processing using threads

Ensure thread safety with synchronization

### File Handling (Bonus)

Save student and course data to files

Load data back into the system
